⚙️ Setup CineVault

To use this app you need a Google OAuth 2.0 Client ID with the Drive API enabled. This is a one-time setup. Your credentials are saved locally in your browser.

Steps:
1. Go to Google Cloud Console
2. Create a project → Enable Google Drive API
3. OAuth consent screen → Add yourself as test user
4. Credentials → Create OAuth 2.0 Client ID (Web application)
5. Add this page's URL as Authorized JavaScript origin
